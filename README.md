# 🎵 OffBeat Music Player

OffBeat Music Player is a modern Android offline music player developed using Java and Android Studio.

## Features

- 🎵 Play local music
- ⏯ Play / Pause
- ⏭ Next / Previous
- 📂 Music Library
- 📃 Playlists
- ⚙ Settings
- 🎚 Playback Speed
- 📱 Beautiful Material UI

## Technologies

- Java
- Android Studio
- XML
- MediaPlayer API
- RecyclerView
